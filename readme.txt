WooCommerce Catalog Enquiry Pro is a catalog plugin - it allows the user to convert a WooCommerce store into a catalog website in a click.

== Description ==
WooCommerce Catalog Enquiry Pro converts your Woostore into a catalog store. You can also use this plugin to get product enquiries from users.

= WooCommerce Catalog Enquiry Pro Features =
Following is a non comprehensive list of FREE features:

* Hide "Add to Cart" button from single product page or any other page (catalog, shop etc.).
* Hide product price for all users or guests (non registered users).
* Admin can allow product selling for selected product categories. Other product categories will be shown in catalog.
* Admin can allow product selling for specific products. Other products will be shown in catalog.
* Admin can restrict the store for selected users. Other users will see catalog only.
* Add a custom button/custom link in single product page or for all products in shop page.
* Choose the group you want to apply catalog settings (registered, non registered users).
* Enable/disable the exclusion rules in a click.
* Responsive product enquiry form.
* Totally customizable form fields.
* Change labeling for enquiry form headers and fields from backend settings.
* Captcha support on enquiry form.
* Fully customizable enquiry button.

= WooCommerce Catalog Enquiry Pro features =
Pro provides more control, more features and an additional support of WC Marketplace, the highest rated multi vendor plugin for WooCommerce.

<strong>Admin can</strong>

* Enquiry Dashboard - Admin will have list of all enquiries and may take neccessary action as per requiremnt.
* Enquiry Cart - Enable user to send enquiries about multiple products.
* Form Builder - Define what information they need from the user by creating their own enquiry form.
* Wide range of email templates to choose from.
* Enquiry Reply from Enquiry Dashboard
* Enquiry Status modify from Enquiry Dashboard
* Create custom email subject (for single product query).
* Allow customers to have a copy of the enquiry via email.
* Define where enquiry button would be positioned.

<strong>Customers can</strong>

* Enquire about multiple product
* Access Enquiry Cart

== Installation ==
1. Upload the `WooCommerce Catalog Enquiry` folder to the `/wp-content/plugins/` directory OR search for "WooCommerce Catalog Enquiry" from your WordPress admin.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Configure the plugin settings through the "WooCommerce Catalog Enquiry settings" menu on your WordPress admin panel.
4. Now click on the settings link and configure the plugin as per your requirement.

This plugin is created for WooCommerce, so you must have to WooCommerce is installed in your WordPress website. 
If your WordPress website haven't WooCommerce installed then first install WooCommerce plugin please follow the link of WooCommerce plugin https://wordpress.org/plugins/woocommerce/. If you are new in WordPress and don't know to how to install a WordPress plugin then follow the link https://codex.wordpress.org/Managing_Plugins. 
You can find how to install WordPress plugin. For this plugin install you can also follow the same link.
WooCommerce Catalog Enquiry maintain the standardization of woo themes. There are we are working on the slandered hooks and filter for the given functionality.
